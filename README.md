# Dream Team Selector

A PyQt5-based fantasy cricket team selector with scoring functionality. Users can create teams, select players based on points, and calculate scores for different matches.

---

## Features

- Create a new fantasy cricket team with a maximum of 11 players.
- Select players from categories: Batsmen (BAT), Bowlers (BWL), All-Rounders (AR), Wicketkeepers (WK).
- Track available and used points while building the team.
- Save and open previously created teams.
- Calculate team scores for multiple matches using player statistics.
- Visual interface built with PyQt5.

---

## Requirements

- Python 3.x
- PyQt5
- SQLite3
